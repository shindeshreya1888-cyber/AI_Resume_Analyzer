# 📄 AI Resume Analyzer

An AI-powered Streamlit web application that analyzes resumes and provides ATS scoring, skill extraction, job match percentage, and visual insights using charts.

---

## 🚀 Features

- 📄 Upload resume (PDF)  
- 🧠 ATS Score estimation  
- 🔍 Skill detection (found vs missing skills)  
- 📊 Job Match Percentage calculation  
- 🥧 Pie chart visualization of skills  
- 📈 Progress bar for job match  
- 🎯 Clean dashboard UI using Streamlit  

---

## 🛠️ Tech Stack

- Python 🐍  
- Streamlit ⚡  
- Matplotlib 📊
